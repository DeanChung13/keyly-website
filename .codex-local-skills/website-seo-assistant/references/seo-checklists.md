# SEO Checklists

Use this reference only when the user needs a deeper audit, remediation checklist, or repeatable optimization workflow.

## Technical SEO Audit

- Confirm important pages return `200` and are not blocked by `noindex`, robots rules, or incorrect canonicals.
- Confirm the XML sitemap exists, is current, and only includes canonical indexable URLs.
- Confirm mobile layout, readability, and tap targets are acceptable on common viewport sizes.
- Check Core Web Vitals and identify the bottleneck category: render blocking assets, image payload, third-party scripts, or layout shift.
- Check internal links for broken destinations, orphaned pages, and weak anchor text.
- Check duplicate or near-duplicate pages created by filters, parameters, or alternate routes.
- Check structured data only where it accurately reflects on-page content.

## On-Page SEO Review

- Write one unique title tag per page with the primary topic near the front.
- Write one compelling meta description that improves click intent without sounding spammy.
- Keep one H1 aligned with the page's primary query intent.
- Use H2 and H3 sections to cover supporting questions and subtopics.
- Improve intro copy so users and crawlers can identify the page topic immediately.
- Add descriptive image alt text where it helps accessibility and topical understanding.

## Content Strategy Review

- Match each page to one dominant search intent: informational, commercial, navigational, or transactional.
- Avoid targeting the same primary keyword across multiple pages unless the intents are clearly different.
- Expand thin pages with evidence, comparisons, examples, FAQs, or decision support.
- Use internal links to connect cluster pages to the core conversion or category page.
- Refresh aging pages when rankings, impressions, or click-through rate decline.

## Backlink and Authority Review

- Prioritize links from relevant sites over random high-volume placements.
- Suggest linkable assets such as original research, calculators, templates, or comparison pages.
- Use digital PR, partnerships, and citations where appropriate for the business model.
- Avoid low-quality link schemes or bulk directory spam.

## Search Console Review

- Compare impressions, clicks, CTR, and average position at page and query level.
- Look for pages with high impressions but low CTR and improve titles and descriptions first.
- Look for queries with positions `8-20` and improve page relevance, section depth, and internal links.
- Look for declining pages and check whether intent mismatch, competition, or technical regressions explain the drop.

## Prioritization Heuristic

Use this order when the user needs a roadmap:
1. Fix indexation and crawl blockers
2. Fix page intent mismatch and weak metadata
3. Improve content depth and clarity
4. Strengthen internal linking and site structure
5. Build authority with relevant backlinks
6. Monitor, measure, and iterate
